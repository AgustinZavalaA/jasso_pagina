¿Que es la arquitectura Cliente-Servidor?

En el modelo cliente servidor interactúan mínimo 2 sistemas informáticos, 1 que controla el usuario (cliente) y otro que responde al usuario (servidor), principalmente consiste en que el usuario haga una petición a una página web o que el usuario realice algún cálculo con datos ingresados por el.

Front-End

¿Qué es el Front-End?

En pocas palabras es lo que ve el usuario, toda la interfaz gráfica así como la experiencia de usuario.

¿Cuáles son los roles dentro del Front-End?

Cumplir con todos los requisitos funcionales de una interfaz gráfica contando en ello su experiencia de usuario, así como mantener una conexión con el backend.

¿Cual es la función de cada uno de los roles del Front End?

Un diseñador de UI se encarga de todo lo que tiene que ver con colores y proporcionar una plantilla Un desarrollador de UX se encarga de trabajar en conjunto con el diseñador para hacer una aplicación sencilla de usar. Un desarrollador de frontend se encarga de darle cierta programabilidad a las funcionalidades que los diseñadores le ponen en la plantilla.

¿Cuáles son los principales lenguajes y frameworks usados en el Front- End?

    javaScript Python C# CSS React.js Angular.js

BackEnd

¿Qué es el backEnd?

se encarga de diseñar la lógica y las soluciones para que todas las acciones solicitadas en una página web sean ejecutadas de manera correcta.

¿Cuáles son los roles del BackEnd?

BackEnd developer y Database administrator.

¿Cuál es la función de cada rol?

BackEnd developer se encarga de hacer la conexión entre la vista generada por el frontend con las funcionalidades del backend Database administrator administra la conexión a la/las bases de datos que requiera el sistema.

¿Cuáles son los principales lenguajes de programación y frameworks del backend?

    Python JavaScript Django PHP C#

¿Qué es un Stack?

Es una pila donde el último entrar es el primero en salir. como por ejemplo los artículos en un supermercado.
